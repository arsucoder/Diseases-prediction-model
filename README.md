# Multiple Disease Prediction Model using Machine Learning

## Project Overview

The **Disease Prediction Model** is a machine learning-based healthcare application designed to predict the likelihood of multiple diseases from user-provided medical information. The project aims to support early disease detection by using trained classification models for different medical conditions.

**Live Demo:** [Disease Prediction Model on Streamlit](https://diseases-prediction-model.streamlit.app/)

## Features

* Predicts the risk of multiple diseases using machine learning.
* Supports prediction for:

  * Heart Disease
  * Diabetes
  * Parkinson's Disease
  * Breast Cancer Disease
  * *(Additional diseases will be integrated.)*
* User-friendly interface for entering patient data.
* Fast and reliable predictions based on trained models.
* Deployed and accessible through a Streamlit web application.

## Models Used

Different supervised machine learning algorithms were trained and evaluated depending on the disease dataset. The best-performing model for each disease was selected and saved for prediction.

## How to Run

1. Clone the repository.
2. Install the required dependencies:

   ```bash
   pip install -r requirements.txt
   ```
3. Run the application:

   ```bash
   streamlit run app.py
   ```
4. Enter the required medical parameters.
5. View the predicted disease outcome.
